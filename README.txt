Nous avons réalisé le projet Blog.

Pour permettre à l'application de fonctionner, il faut disposer d'une base de données "LOO2", en s'y connectant avec l'utilisateur "LOO" sans mot de passe.

Nous avons mis en place un système de login dans notre application. Pour se connecter, utiliser l'un des profils suivants :
Username : a, Password : pass
Username : b, Password : pass

L'interface est séparée en deux onglets :
- L'onglet Users permet d'avoir la liste de tous les utilisateurs (on ne peut cependant pas voir les mots de passe des autres utilisateurs). L'utilisateur actif peut changer ses propres informations en éditant directement les cases correspondantes.
NB. Dans une application plus professionnelle, il serait sûrement plus judicieux de ne pas voir les informations des autres personnes.
- L'onglet Posts permet d'afficher une liste de tous les messages postés. Il est possible de les filtrer par date et par mots-clés. L'utilisateur peut créer un nouveau message, en précisant un titre, un contenu, des mots-clés, des liens et des images (ces trois derniers étant  séparés par des espaces). Les liens sont des URL, et les images sont des chemins d'accès. Le titre est obligatoire.
L'utilisateur ayant posté le message peut le modifier, voir les informations, et le supprimer.
Ctrl + Espace ou Ctrl + Clic pour supprimer le filtre par mot-clé.
